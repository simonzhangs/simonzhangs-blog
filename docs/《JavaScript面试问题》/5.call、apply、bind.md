# call、apply、bind区别 

call 可以调用函数；
call 可以改变函数中this指向；
call 第一个参数改变指向，第二个参数是传入参数；

```js
function fun() {
    console.log(this.name);
}

let cat = { name: '喵喵'};

fun.call(cat); // 喵喵
```

apply 第二个参数传入参数用数组；

bind 跟call 区别是：bind不会调用函数，函数会作为返回值返回，需要自己调用。
